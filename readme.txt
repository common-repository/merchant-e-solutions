The easy to use payment gateway plugin for Merchant E-Solutions. The plugin is fully certified and endorsed by Merchant E-Solutions.

== Description ==

If you're using Merchant E-Solutions and want it to work with WooCommerce this is the plugin for you. Looking for a  easy to use plugin that works without fuss? Merchant e-Solutions is for you! Get started in minutes and start increasing your webshops revenue today!

**Features**

* Easy to use dashboard
* Automated callback for asynchronous updates
* Order number prefixes
* Configurable Title and Description

**Security**

* PCI Compliant! No credit card data stored locally
* Forces SSL verification on the payment gateway
* Session timeout for remote gateway
* Local SSL Supported but not required

== Installation ==

Simply click install from your dashboard and the plugin will be autmatically downloaded.
Once activated, you can configure Merchant e-Solutions in the WooCommerce Checkout dashboard page.

= Minimum (Website) Requirements =

* WordPress 4.0 or greater
* PHP version 5.3 or greater
* WooCommerce 2.3 or greater

= Other Requirements =

* Merchant e-Solutions account
